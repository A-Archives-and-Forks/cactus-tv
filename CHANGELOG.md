# v1.1.0 Cactus Performance Core

- 首页首屏不再等待 D1 片单同步和 `/api/health`。
- 新增本地轻量推荐引擎：继续观看、类型/题材/年代偏好、完成比例、收藏和多样性重排。
- 新增个性化推荐开关；推荐计算不产生额外网络请求。
- 首页卡片新增推荐原因和观看进度条。
- 手势增加横向拖动精细定位，亮度与音量偏好可在当前设备记忆。
- 保留 Cactus Clean Stream，可关闭；Streamflow 继续彻底停用。
- 播放器仍使用 hls.js / dash.js / 原生 video，维持 200/300 秒目标缓冲策略。

# v1.0.0 Cactus Player 2.0

- 保留自研播放器外壳和 hls.js / dash.js 内核。
- 手机和平板目标前向缓冲 200 秒，桌面目标 300 秒。
- DASH 同步采用长缓冲配置。
- 增加 HLS FPS 降级保护、真实码率修正和紧急码率降级。
- 重写触控方向锁、连续双击、非线性横向拖动、长按倍速和手势反馈。
- 增加 Cactus Clean Stream 实验性 HLS 广告分片过滤。
- 数据源媒体白名单支持安全的 `*.example.com` 通配符。
- 主动 Streamflow 预取继续保持停用，不与正常播放争抢资源。
