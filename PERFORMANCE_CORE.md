# Cactus Performance Core v1.1.0

这一版不删除基础功能，也不重新引入 Streamflow。目标是：让首页先可用、播放器保持完整手势、推荐逐渐贴合使用者，同时不让推荐计算阻塞首屏。

## 首页性能

- 首页播放器代码继续按需加载。
- 首屏不再等待 D1 片单同步和 `/api/health` 完成。
- 首页数据返回后立即渲染；D1 同步和站点状态改为空闲阶段加载。
- 横向片单按屏幕范围分批创建卡片，屏幕外图片继续使用 IntersectionObserver 延迟加载。
- 推荐计算只遍历首页候选和最多 200 条历史记录，不发起额外网络请求。

## 个性化推荐

首页最多增加两行：

- `继续观看`：按最近观看时间排序，仅保留尚未接近看完的内容。
- `为你推荐`：综合收藏、完成比例、观看时长、类型、题材、年代、评分、热度和少量探索权重。

推荐采用多样性重排，避免同一种类型或同一年代连续霸屏。算法全部在当前浏览器内轻量运行，可在播放设置中关闭。

## 播放器

- 保留 hls.js / dash.js / 原生 video 三条播放路径。
- 保留手机和平板约 200 秒、桌面约 300 秒的目标缓冲，并受最大内存字节数限制。
- 保留 Cactus Clean Stream，仍可单独关闭。
- Streamflow 维持删除状态，不产生主动预取请求。

## 手势

- 单击显示或隐藏控制栏。
- 左右双击快退/快进，中间双击播放或暂停。
- 左侧上下调亮度，右侧上下调音量。
- 横向滑动预览进度；向上或向下偏移后自动进入精细定位模式。
- 长按临时 2 倍速，松手恢复。
- 亮度和音量偏好保存在当前设备。
- 锁定后停止进度、亮度、音量和长按手势。
